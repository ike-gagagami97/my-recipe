# Product vision

## 一言で

**My Recipe** — レシピを保存・閲覧・追加できるシンプルな Web アプリ。

## 目的

個人または少人数が、日常の料理レシピを素早く記録し、後から探しやすくする。

## 現状（2026-07）

- Next.js 16 + Supabase + Vercel の土台のみ
- ランディング画面のみ表示（CRUD 未実装）
- DB スキーマ未定義（`supabase/migrations/` は空）

## 想定ユーザー

- 自分用にレシピを溜めたい個人
- 将来的に共有・認証を足せる余地は残すが、**最初のスコープでは認証なし**を想定

## 非スコープ（当面やらない）

- SNS 連携・ソーシャルフィード
- 栄養計算・カロリー管理
- ネイティブアプリ
- 多言語対応（UI は日本語または英語のどちらかに寄せる）

## 成功の定義（MVP）

- ローカルまたはホストされた Supabase にレシピを保存できる
- 一覧 → 詳細 → 追加の一連の流れがブラウザで完結する
- RLS / grant が正しく、anon から意図した操作だけできる
