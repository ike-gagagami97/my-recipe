```markdown
# NNNN. Title

- Status: proposed | accepted | superseded
- Date: YYYY-MM-DD

## Context

なぜ判断が必要か。

## Decision

何を選んだか。

## Consequences

良い影響 / 悪い影響 / フォローアップ。
```
